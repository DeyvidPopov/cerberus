# Keystroke-dynamics detector comparison

Generated by `npm run eval:keystroke --workspace @cerberus/server` (ADR-0010).
Reproducible: fixed seed `20240601`; re-running yields identical numbers.

- **Protocol:** Killourhy & Maxion (2009). Per subject: train on the first
  200 genuine reps; test on the held-out genuine reps + the
  first 5 reps of every other subject.
- **Subjects:** 51 · **Feature dimension:** 31
  (position-indexed hold/DD/UD, the shared M6 extractor).
- **Dataset:** `docs/evaluation/data/DSL-StrongPasswordData.csv` (NOT committed — real human
  captures, PROJECT.md §5). At the equal-error operating point FAR = FRR = EER.

_K&M 2009 column: the originally published EER for the two classic detectors
(Mahalanobis ≈ 11.0%, one-class SVM ≈ 10.2%); isolation forest is a later addition
(ADR-0002) with no figure in that study. Small differences from published numbers
are expected (regularization choice, EER interpolation, implementation detail)._
